# Restaurant Menu & Order Management System

This project is a Python-based Restaurant Order Management System built using core data structures like lists and dictionaries.

## Description

The program simulates how a restaurant manages its menu, customer orders, inventory, and sales data. It performs various operations such as displaying menu items, handling cart operations, updating stock, and analyzing sales records.

## Features

- Display menu grouped by categories
- Add, update, and remove items from cart
- Generate order summary with GST calculation
- Track inventory and show reorder alerts
- Analyze daily sales and find best-performing day
- Identify most ordered items
- Print all orders in a structured format

## Concepts Used

- Lists and Dictionaries
- Nested Data Structures
- Loops and Conditionals
- String Formatting
- Deep Copy (copy module)
- Basic Data Analysis using Python
